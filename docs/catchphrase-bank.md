### קול נהוראי — חירטוטים

> נשמע עמוק. לא תמיד. זה ה-feature.

| משפט |
|---|
| אנשים לא רגילים לראות דברים רגילים. |
| הפתרון תמיד היה שם. שאלת בדיוק בזמן. |
| בסוף, כולנו מחפשים את אותו דבר. |
| זה לא קוד. זה בניית עולם. |
| הכי חכם זה לדעת מה לא יודעים. |
| אחי, זה לא באג. זה feature שמחכה שישאלו אותו. |
| כשאתה מבין את זה — אתה מבין הרבה דברים. |
| לפעמים התשובה הכי פשוטה היא הכי נכונה. |
| מי שלא שאל — לא יקבל. זה בחיים, זה גם בקוד. |
